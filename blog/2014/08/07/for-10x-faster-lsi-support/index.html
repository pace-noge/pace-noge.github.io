For 10x Faster Lsi Support

Aug 7th, 2014

Mungkin blogger octopress sering melihat notifikasi di bawah saat menjalankan perintah rake deploy.

Apabila para blogger perlu memproses banyak file (membuat postingan baru), maka lsi ini berguna untuk meningkatkan kecepatan generate file oleh jekyll. Pada notofikasi terlihat alamat web dari gsl, Ketika saya mencoba membuka alamat tersebut chrome menampilkan pesan This webpage is not available. Alamat ruby gsl adalah rb-gsl. Disini saya akan menjelaskan cara menginstall Gsl pada OsX mavericks 10.9.4.

Langkah-langkah nya adalah:

  1. menginstall gsl dengan brew
  2. kemudian menginstall rb-gsl

install gsl library menggunakan brew

install rb-gsl dengan menggunakan gem install
