Konfigurasi Octopress

Aug 3rd, 2014

Konfigurasi dari octopress berada dalam file yang _config.yml.

agar lebih mudah terindex oleh google, tag author sebisa mungkin sama dengan nama G+, untuk description di kosongkan saja karena akan terisi secara otomatis bila generate post.

subscribe_rss: /atom.xml subscribe_email: email:

paginate: 10 # Posts per page on the blog index paginate_path: “posts/:num” # Directory base for pagination URLs eg. /posts/2/ recent_posts: 5 # Posts in the sidebar Recent Posts section excerpt_link: “Read on →” # “Continue reading” link text at the bottom of excerpted articles excerpt_separator: “”

titlecase: true # Converts page and post titles to titlecase

default_asides: [asides/recent_posts.html, asides/github.html, asides/delicious.html, asides/pinboard.html, asides/googleplus.html]

github_user: pace-noge github_repo_count: 0 github_show_profile_link: true github_skip_forks: true

twitter_user: udhent

google_plus_one: true google_plus_one_size: medium

googleplus_user: 118040098239619744268 googleplus_hidden: false

pinboard_user: pinboard_count: 3

delicious_user: delicious_count: 3

disqus_short_name: disqus_show_comment_count: false

google_analytics_tracking_id: UA-XXXXXX-X

facebook_like: false “`

Konfigurasi diatas untuk social media

  • github_user (username github)
  • twitter_user (username twitter)
  • googleplus_user (biasanya angka dari halaman profile di G+)

untuk konfigurasi yang lain sama saja, berisi username, kemudian untuk google_analytics_tracking_id bisa di dapatkan dari Google Analytics
