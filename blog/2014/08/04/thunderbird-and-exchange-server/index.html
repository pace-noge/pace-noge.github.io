Thunderbird and Exchange Server

Aug 4th, 2014

instalasi Davmail

Untuk dapat melakukan koneksi dari thunder bird yang dibutuhkan adalah Davmail sebagai perantara antara Thunderbird dan exchange server.

Untuk instalasi davmail anda dapat mendownload dari sini Davmail. setelah anda mendownload versi terakhir dari davamail, buka terminal kemudian masuk ke directory tempat download davmail tersebut. kemudian masukkan perintah berikut untuk instalasi.

atau bisa dilakukan penginstalan melalui ubuntu software center, kemudian cari aplikasi dengan nama davmail. Davamail membutuhkan java agar dapat berjalan dengan normal.

setelah terinstall buka davmail melalui logo ubuntu kemudian ketik davmail. biarkan pengaturan lainnya tetap standar, yang perlu anda rubah adalah field untu (OWA) exchange server. Anda dapat mnegisinya dengan alamat owa dari exchage server anda, misalnya https://webmail.nayaklabs.com/owa atau https://webmail.nayaklabs.com/exchange.asmx kemudian save,

disini saya asumsikan bahwa anda telah mendownload thunderbird, apabila belum maka anda dapat mendownload thunderbird dari ubuntu software center.

Tahap selanjutnya adalah konfigurasi thunderbird

langkah lengkah pengaturannya adalah:

  1. Buka thunderbird
  2. klik pada menu File -> New -> Existing Mail account …
  3. ISi nama, email, dan password

  4. Pilih manual configuration dengan pengaturan dibawah bila anda tidak merubah stingan pada davmail.

    • Incoming:

      1. Type: IMAP
      2. Server: localhost
      3. Port: 1143
      4. SSL: None
      5. Authentication: normal password

    • Outgoing:

      1. Type: SMTP
      2. Server: localhost
      3. Port: 1025
      4. SSL: None
      5. Authentication: normal password

    • username: isi dengan username owa anda

  5. Klik test apabila sudah berhasil anda dapat meng-klik Done

pengaturan selesai, pada saat ini thunderbird sudah dapat melakukan koneksi pada exchange server anda.
